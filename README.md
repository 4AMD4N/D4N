# 易世界

在《易世界》中，万物都由阴与阳构成。

从天地运行到四季更替，从雷霆骤起到流水奔涌，世间的一切变化，都源于最初的两种力量——阴与阳。

它们彼此交融，彼此转化，演化出八卦，衍生出万象。

而你，将成为变化的掌控者。

每一局游戏，你都将从最基础的阴阳爻开始，通过抽取、组合与变爻，构筑属于自己的卦象体系。看似简单的一次变化，可能让平凡的卦象蜕变为强大的组合；一次关键的变爻，也可能彻底改变整局游戏的走向。

在这里，没有固定的套路。

因为每一个卦，都能够变化。

每一次变化，都可能诞生新的可能。

游戏玩法

游戏采用短局制 Roguelike 卡牌玩法。

每局游戏由多个回合组成，玩家需要在有限的资源与机会下，不断构筑更强大的卦象组合，以达成关卡目标并击败最终 Boss。

阴阳生八卦

游戏中最基础的资源只有两种：

阳爻 ⚊
阴爻 ⚋

通过不同排列组合，你将形成八种不同的基础卦象：

乾、坤、震、巽、坎、离、艮、兑。

每一种卦象都拥有独特能力与得分方式。

变爻系统

变化，是《易世界》的核心。

你可以在关键时刻改变卦中的某一爻，将原本的卦象转变为新的形态。

同样的起点，可能演化出完全不同的结果。

选择保留现状，还是赌上一次变化的机会？

每一次决定，都可能影响最终结局。

命格构筑

在冒险过程中，你将获得各种命格。

命格会永久改变游戏规则：

强化特定卦象
增加变爻次数
改变抽牌概率
解锁特殊组合

不同命格之间还能产生联动效果，构筑出截然不同的流派。

发现与收集

从八卦开始，你将逐渐发现更多隐藏组合、特殊变化与神秘路线。

有些卦象唾手可得。

有些则需要特定条件才能出现。

而有些秘密，也许需要无数次轮回才能揭开。

抽取阴阳，组成卦象。

改变一爻，改写命运。

欢迎来到《易世界》。
