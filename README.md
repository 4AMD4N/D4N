# 易世界

《易世界》是一款基于阴阳、八卦与变爻机制的短局回合制卡牌游戏。

## GitHub Pages 发布方式

1. 创建一个新的 GitHub repository，例如 `yishijie`
2. 上传本文件夹内的 `index.html`、`styles.css`、`game.js`
3. 进入 repository 的 `Settings` -> `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，文件夹选择 `/root`
6. 保存后等待 GitHub Pages 生成链接

项目链接通常会是：

```text
https://你的用户名.github.io/yishijie/
```

## 后续更新

以后更新游戏时，只需要替换这三个文件：

- `index.html`
- `styles.css`
- `game.js`

GitHub Pages 会自动更新同一个网页链接。
